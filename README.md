# Omago – CVE Intelligence Assistant

A Streamlit-based cybersecurity dashboard for CVE analysis, built with a RAG architecture using Llama 3.1-70B via Groq, NVD, CISA KEV, and Exploit-DB.

## Features
- Home chat interface with Omago AI assistant
- Dashboard with live CVE visualizations
- CVE Lookup by ID
- Chat history management
- Analysis panels for model and dataset evaluation

## Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/omago-cve-assistant.git
cd omago-cve-assistant
pip install -r requirements.txt
streamlit run app.py
```
